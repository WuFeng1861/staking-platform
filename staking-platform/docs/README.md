# 质押平台 API 文档

## 基本信息

- 基础URL: `http://localhost:62777`
- 所有接口返回格式统一为：

```json
{
  "code": 0,        // 0表示成功，非0表示错误
  "data": {},       // 实际返回数据
  "message": "操作成功", // 提示信息
  "timestamp": 1628000000000 // 时间戳
}
```

## 错误码说明

| 错误码 | 描述 |
|-------|------|
| 0 | 成功 |
| 400 | 请求参数错误 |
| 401 | 未授权（管理员密码错误） |
| 404 | 资源不存在 |
| 409 | 资源冲突（如交易hash已存在） |
| 500 | 服务器内部错误 |
